**************************************************************************************
Project Features :-
**************************************************************************************
1 - A DBMS System designed with a simple , readable and basic linux commands and
    utilities.
2 - The shell script is actually divided into separated sub-scripts to ease the
    process of development .
3 - The DBMS handles PK constraints (unique - not null).
4 - Column names can not be duplicated.
5 - The DBMS displays data columns in good format .
6 - each table is represented by two files of csv formats , one for raw data.
    and the other for meta data .
7 - Data update feature by primary key (bonus). (already done)
8 - DBMS Accepts data types for each data field (bonus). (already done)
9 - All input data by user are validated .
10 - A primary key can not be duplicated or updated (Data integrity costraint).
